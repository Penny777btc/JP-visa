# JP-Visa Knowledge Base 🇯🇵

日本签证路径知识库：从「技術·人文知識·国際業務」→「高度専門職」→「永住者」的完整申请指南。

## Features

- 📊 完整的高度人才积分计算表
- 📋 每个阶段的必备文件清单
- ⚠️ 常见拒签原因分析与应对策略
- 🔍 实时搜索/关键词过滤
- 📱 响应式设计，支持移动端访问
- 🌙 Premium 暗色主题

## Data Sources

所有信息来源于日本出入国在留管理厅（ISA）官方网站：
- [ISA 官方网站](https://www.moj.go.jp/isa/)
- 永住許可ガイドライン（令和8年2月24日改订）
